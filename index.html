Exercícios Introdução JavaScript

O trabalho consiste em criar diversas ativdades para testar
os conhecimentos de Java Script adquiridos em sala de aula.

Cada titilo das atividades apresentadas contem um link especifico
que o levará a sua respectiva pagina.

Aluno Aprovado

 atividade consiste em:

Criar um programa em Java Script onde o usuário irá digitar uma nota nete 0 e 10.

A saida das notas deverão apresentar os seguintes resultados:

- Notas abaixo de 4,5 "CONCEITO F".
- Notas entre 4,5 e 4,9 "CONCEITO D".
- Notas entre 5,0 e 6,9 "CONCEITO C".
- Notas entre 7,0 e 8,9 "CONCEITO B".
- Notas entre 9,0 e 10,0 "CONCEITO A".

Média do aluno

 atividade consiste em:

Criar um programa em Java Script que Recebeba três notas de uma diciplina. E retorne a média das notas.

O sistema deve escrever a média do Aluno.
O respectivo nome do aluno.
O conceito final.

O conceito final: - Médias abaixo de 4 "REPROVADO".
- Médias entre 4,1 e 5,9 "RECUPERAÇÃO FINAL".
- Médias acima 6 "APROVADO".

velocidade média

 atividade consiste em:

Criar um programa em Java Script para calcular a velocidade média, em km/h, de um veículo.

A fórmula da velocidade média é: vm = e/t, onde:

vm = velocidade média (em km/h);
e = espaço percorrido em km;
t = tempo, em horas, para percorrer aquele espaço;

preço final

 atividade consiste em:

Criar um programa em Java Script que calcule o preço final de um produto dada as seguintes condições:.

- Se o pagamento for à vista, tem desconto de 5%;
- Se o pagamento for parcelado em 3 vezes deve ser dado um acréscimo de 10% no total a ser pago;
- Se o parcelamento for em 5 vezes, o acréscimo será de 20%.
O programa deve mostrar o valor final do produto. Se for à vista o valor de cada prestação.

Farenheit para Celsius

 atividade consiste em:

Criar um programa em Java Script que receba a temperatura em graus Farenheit para o usuário, transforme e mostre a temperatura em graus Celsius.

Fórmula: Celsius = (5 * (Farenheit -32) / 9).

Celsius para Farenheit

 atividade consiste em:

Criar um programa em Java Script que receba a temperatura em graus Celsius, transforme e mostre em graus Farenheit.

Fórmula: Farenheit = Celsius × 1,8 + 32.

álcool ou gasolina

atividade consiste em:

Criar um programa em Java Script receba o valor da gasolina e do álcool de um posto e diga qual é o combustível mais vantajoso abastecer.

Sabendo que somente é vantagem abastecer álcool se o preço do mesmo é menor ou igual a 70% do valor da gasolina.

imc

 atividade consiste em:

Criar um programa em Java Script que calcule o IMC - Índice de Massa Corporal de uma pessoa e depois mostre em qual categoria (veja tabela abaixo) esta pessoa se encontra.

Fórmula de cálculo do IMC = p/(a²), onde:
p = peso em kg;
a = altura em metros.

Tabela de saúde:
Até 18,5 - Abaixo do peso
De 18,5 - 24,9 - Saudável
De 25,0 - 29,9 - Sobrepeso
De 30,0 - 39,9 - Obeso
Acima de 40 - Extremo ou alto risco de obesidade
